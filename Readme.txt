All files have the following purposes:

- Dataset creation full.ipynb: Scrapes the data from the internet, preprocesses and reformats it into two different files.
- combined_preprocessed.csv: The first data file, mostly for testing uses.
- set2.csv: The data file used by the dashboard.
- hero_list.py: A list of towns and heros used by the dataset creation full file 
- main.py: Running this file will create the dashboard at http://127.0.0.1:8050/
- functions.py: This file serves as a library used by main.py
